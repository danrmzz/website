Weather App

This Weather App is a Java based application that provides users with real time weather information for a specified location. It fetches weather data from an external API and displays it in a GUI. Users can enter a location, and the app retrieves and presents weather details, including temperature, weather condition, humidity, and wind speed.

Ban Tracker

An app that monitors player bans on the Minecraft SaicoPvP server. It scrapes data from over 100 pages, processing more than 1,000 entries. The app counts bans, fetches player UUIDs via the Mojang API, and displays the top 10 most banned players through a Flask web interface, providing detailed ban statistics.

MIPS Simulator

A core part of a mini processor simulator called MySPIM, developed using C on a Unix platform. MySPIM will demonstrate functions of the MIPS processor as well as the principle actions of the datapath and control signals. The simulator is designed to provide a detailed understanding of how instructions are processed and executed within the MIPS architecture.

Snake Game

Classic Snake game created using the Tkinter library for the GUI. The code handles snake movement, food generation, and collision detection. It also features a scoring system to track the player's progress and a game over screen that appears when the snake collides with itself or the game boundaries.

Queue Simulation

A queue management system developed for a theoretical movie ticketing scenario, aiming to optimize customer service. The system utilizes a linked list structure for ticketing operations with O(1) efficiency, significantly reducing operational latency. An algorithm for dynamic customer allocation was engineered, greatly optimizing booth operations.
